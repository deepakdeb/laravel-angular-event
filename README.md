# Laravel

## Installation

Clone this git repo, copy env from env.example and run these commands

```
composer install
```

Then run the migrations.

```
php artisan migrate
```
run the artisan command.

```
php artisan serve
```


# Angular


Go to the angular root folder with the command

```
cd resources/event-angular/app
```

Install Dependancy

```
npm install
```

Run Angular 

```
npm start
```
